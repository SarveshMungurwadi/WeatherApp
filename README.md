# WeatherApp
Weather Web Application is an real-time web application which shows current weather of an
city. This web application is built using Node.js and openweathermap API
## Technologies used:
1.HTML5 <br />
2.CSS <br />
3.Javascript <br />
4.Node.js <br />
5.Express.js <br />
and Openweathermap API
